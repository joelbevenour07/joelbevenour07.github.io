DegreeBase - 2024

View on GitHub

Account based class scheduler with real college class database. Utilizes MySQL, SQLAlchemy, Flask, Jinja2, and HTML to create web page that allows users to create a new account, choose a major, and fill out a weekly class schedule visualiser by browsing and enrolling in courses.

Login

Basic user login page with choice to register as a new user.

Student Registration

User registration page where new users enter their name, email, and some basic student info like their current grade level and credits already completed to calculate remaining credits later.

User Dashboard

New Users are shown their current data including their chosen major, remaining credits, grade level and more. They are given 4 options at the bottom: View All classes, view major specific classes, their enrolled courses, and a visual scheduler.

User Specific Classes

Database relates certain classes to specific majors and their required courses, and you can browse those courses and all their sections, listing all the times and days of the week that class is running.

User Enrolled Classes

Allows users to view their enrolled courses after choosing.

Scheduler

Admin Page Student Search

Admin Full Db

Admin Full Db Cont.

Dashboard with PM5

Main RowPal screen which allows you to turn on bluetooth to find activated PM5 Concept 2 Rowing Machines. You simply tap the rower id that matches yours and the screen on your rowing machine will confirm connection.

Data screen for distance

Various rowing statistics, some of which are not usually displayed on a PM5 rowing machine, to be displayed during a rower's workout. Configured to display the remaining distance of a workout when you choose single distance.

Data screen for time

Same type of rowing statistic screen as before, configured to display time remaining when a user selects single time.

Just Row Option

Again, similar data screen but configured for the "Just Row" option on a Concept 2 Rower. Displays user's distance rowed so far in meters and current time elapsed.

RowPal - 2025

View on GitHub

A Concept 2 Rowing Machine Data display which can connect to PM5 Rowing screens via Bluetooth. Displays internal statistics regarding current Concept 2 Rowing workouts that are normally hidden from the user with the typical PM5 interface in real time. Utilizes the Concept 2 SDK.
